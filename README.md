# LaptopPricePrediction

I built this project for a Data Mining and Machine Learning exam. The goal was to take a laptop-price dataset from Kaggle, inspect it, clean it, preprocess it, and train a predictive model able to estimate laptop price from hardware-related features.

## What I Did

In the notebook I worked through the full pipeline:

- dataset inspection,
- data cleaning,
- preprocessing,
- feature handling,
- model training,
- model evaluation.

The repository contains the notebook together with the report and the slides used to present the project.

## Repository Contents

- `DMML_Project.ipynb`: the main notebook.
- `DMML_Project.pdf`: written report.
- `DMML_Project.pptx`: presentation slides.

## How To Use It

1. Create a Python virtual environment.
2. Install Jupyter Notebook or JupyterLab.
3. Install the Python dependencies used in the notebook.
4. Provide the expected Kaggle dataset file at `dataset/laptopData.csv`.
5. Open and run `DMML_Project.ipynb`.

## Notes

- The dataset itself is not currently stored in the repository, so local reproduction requires adding `dataset/laptopData.csv`.
- The PDF report and slides are included because this was produced as a full academic project, not just as a standalone notebook.
